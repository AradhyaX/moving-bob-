# Move Bob

A simple web project where you can interactively move a character named Bob using the `ArrowLeft` and `ArrowRight` keys on your keyboard!

## Technologies used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Play
1. Open up `index.html` in your browser.
2. Click on the screen to focus.
3. Use your left and right arrow keys to move Bob around!
